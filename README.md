# 🎮 Tic Tac Toe Game

A simple and interactive **Tic Tac Toe** web game built using **HTML, CSS, and JavaScript**.
This project demonstrates core front-end development skills including DOM manipulation, game logic, and responsive UI design.

## ✨ Features

* 🧠 Two-player gameplay (Player X vs Player O)
* 🎨 Clean and modern user interface
* ⚡ Fast and responsive design
* 🔄 Game reset functionality
* 📱 Works on desktop and mobile devices

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & layout
* **JavaScript (Vanilla JS)** – Game logic & interactivity

## 🧩 How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/Tik-Tak-Tok.git
   ```

2. Open the project folder.

3. Run `index.html` in your browser.

✅ That's it!

## 🌐 How to Deploy (GitHub Pages)

1. Go to **Settings** of this repository
2. Open **Pages** section
3. Select branch → `main`
4. Save and get your live link 🎉
